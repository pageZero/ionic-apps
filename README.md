## appDemo_2_wechat
仿微信客户端，有微信/消息这两页，使用了tab+手势识别，自定义服务，自定义指令，添加了一些样式，跟着大神的博客写的。

功能：
* 消息列表
* 联系人列表
* 消息置顶/取消置顶
* 标记消息已读/标记未读
* 聊天详情页
* 可伸缩输入框
* 下拉刷新
